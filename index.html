Building a Personal Dashboard.

Building a personal dashboard is easy with tonic. Easily aggregate data from disparate APIs and generate a static website on a schedule.

Getting Started.

Before you get started, you need to install node.js and npm.

Install tonic.

$ npm install -g tonic

Creating Your First tonic Website.

$ tonic new MyFirstTonicWebsite

You will now have a brand new tonic website in the directory MyFirstTonicWebsite. This website is a skeleton of a tonic website that will update a page every minute with a really bad pun.

Running Your tonic Website.

$ tonic run MyFirstTonicWebsite

Viewing Your tonic Website

Open the index.html file in the directory MyFirstTonicWebsite/public/index.html.

While tonic is running, this page will be updated every minute with a new joke.

Customizing Your tonic Website.

Jobs.

A job is a small piece of JavaScript which will execute periodically to get data from an API.

This job will run every 20 minutes. Once your logic has completed, pass your data into the done callback so that it can be displayed on your site.

job('my_new_job', '20 min', function(done) {
    // Logic to get data
    done(data);
});

Why Static Pages?

The goal of tonic.js is to be easy to setup and easy to run. By generating static pages, tonic allows users to forget about setting up an application server or configuring a database. All you have to do is run tonic run from the command line and your site is ready to go.

Tonic can be run on your own server, Heroku, or Azure.

Most of the design inspiration for this project was based on the principles of other software such as Jekyll, Wintersmith, and Dasher.

Rather Drive Stick?

If you prefer to not use the automatic generator, push a branch named gh-pages to your repository to create a page manually. In addition to supporting regular HTML content, GitHub Pages support Jekyll, a simple, blog aware static site generator written by our own Tom Preston-Werner. Jekyll makes it easy to create site-wide headers and footers without having to copy them across every page. It also offers intelligent blog support and other advanced templating features.
